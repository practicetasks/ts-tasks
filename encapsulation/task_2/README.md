## Задача

Создайте класс Car, который представляет автомобиль.

Класс должен использовать принципы инкапсуляции: поля должны быть закрыты от прямого доступа извне, а работа с ними должна происходить через методы.

### Поля класса

- make: string — марка автомобиля
- model: string — модель автомобиля
- year: number — год выпуска
- speed: number — текущая скорость

Все поля должны быть приватными.

---

## Конструктор

Создайте конструктор, который принимает:

`make: string model: string year: number`

Начальная скорость автомобиля должна быть равна 0.

---

## Геттеры и сеттеры

Для полей make, model и year создайте геттеры и сеттеры.

Для поля speed создайте только геттер, чтобы скорость нельзя было изменить напрямую извне класса.

---

## Правила валидации

### make

Марка автомобиля не может быть пустой строкой.

Если передано некорректное значение, вывести:

`Марка не может быть пустой.`

### model

Модель автомобиля не может быть пустой строкой.

Если передано некорректное значение, вывести:

`Модель не может быть пустой.`

### year

Год выпуска должен быть не меньше 1900 и не больше текущего года.

Если передано некорректное значение, вывести:

`Некорректный год выпуска.`

---

## Методы

### accelerate(increment: number): void

Метод увеличивает текущую скорость автомобиля на переданное значение.

Если значение меньше или равно 0, скорость не меняется, а в консоль выводится:

`Значение ускорения должно быть положительным.`

Если значение корректное, увеличить скорость и вывести:

`Автомобиль ускорился на <increment> км/ч.`

---

### brake(decrement: number): void

Метод уменьшает текущую скорость автомобиля на переданное значение.

Если значение меньше или равно 0, скорость не меняется, а в консоль выводится:

`Значение замедления должно быть положительным.`

Скорость не должна становиться отрицательной.

Если значение корректное, уменьшить скорость и вывести:

`Автомобиль замедлился на <decrement> км/ч.`

---

### printCarInfo(): void

Метод выводит информацию об автомобиле:

`Марка: <make> Модель: <model> Год выпуска: <year> Текущая скорость: <speed> км/ч`

---

## Пример использования

```ts
const car = new Car("Toyota", "Camry", 2020);
car.printCarInfo();

car.accelerate(50);

car.printCarInfo();

car.brake(20);

car.printCarInfo();

car.setYear(1800);
car.setMake("");

car.accelerate(-10);
car.brake(-5);

car.printCarInfo();

car.setYear(2018);
car.setMake("Honda");
car.setModel("Accord");

car.printCarInfo();
```
